
# Calculator App

- Java 17
- JUnit 5
- Maven 3.8.6
- OpenAPI 3.0.3
- Vert.X 4.3.4

## Building

1. **Test** To launch your tests:

```
./mvnw clean test
```

2. **Package** To package your application:
```
./mvnw clean package
```

3. **Run** To run your application:

```
./mvnw clean compile exec:java
```
